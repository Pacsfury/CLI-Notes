# CLI-Notes
A fast C program to save your notes.

! DONT USE THIS ! It has just been started

---

Welcome to Notes, where you will be able to save notes fast and lightweight.

This is a CLI tool, so it will be run in your terminal.

### How to run?

1. Compile or get the `.exe` file
2. Using your Command Prompt, enter where the `.exe` file is:
`cd PATH_TO_THE_FOLDER`
3. Everything ready!

### How to use?
There are different commands. Right now, there are only two, but I am adding new every time I can.
1. `new`:
To create a new note, use the following structure:

```powershell
./app.exe new --title yournotename
```
2. `open`:
To see a note title and content, use the following structure:

```powershell
./app.exe open --title yournotename
```
